LWC Fundamentals

- What is LWC? 

        
        • Lightning Web Components (LWC) are custom HTML elements built using HTML and modern JavaScript.
        • Lightning Web Components (LWC) uses core Web Components standards and provides only what’s necessary to perform well in browsers supported by Salesforce. Because it’s built on code that runs natively in browsers, Lightning Web Components is lightweight and delivers exceptional performance. Most of the code you write is standard JavaScript and HTML.
        • Lightning Web Components (LWC) are a fundamental building block in Salesforce's Lightning Platform for building user interface (UI).

- Please create a component that shows "Hello World" text on UI.

         // We can create a component by right clicking on LWC folder or we can use the code below 
        

        sfdx force:lightning:component:create --type lwc -n helloWorld
        

         // This will create a folder named "helloWorld" with necessary files. 
        
         <!-- helloWorld.html --> 
        

        <template>
          <p>Hello, World!</p>
        </template>
        
  
         // helloWorld.js 
        

        import { LightningElement } from 'lwc';
        
        export default class HelloWorld extends LightningElement {}
        

        
            // We can use it in other Lightning components or in the Lightning App Builder by adding it to a Lightning Page.
            // To use it in a Lightning App Page, go to Setup, find the App Page you want to edit, and add your "helloWorld" component to the page.
            // When we view the Lightning App Page, we'll see our "Hello, World!" message displayed. This is a simple example, but we can build more complex components.

- Please create a component that shows "Hello World" text with JavaScript on UI.

         // a simple example of an LWC that displays a "Hello, World!" message: 
        
         <!-- helloWorld.html --> 
        

        <!-- helloWorld.html -->
        <template>
          <p>{greeting}</p>
        </template>
        
  
         // helloWorld.js 
        

        // helloWorld.js
        import { LightningElement } from 'lwc';
          
        export default class HelloWorld extends LightningElement {
          greeting = 'Hello, World!';
        }
        

        
            // In this example, we have an LWC named "HelloWorld."
            // The HTML file defines the template, and the JavaScript file uses the LightningElement base class to create the component.
            // The greeting property is used to display the message in the template.
            // We can then include this LWC in our Salesforce Lightning App to display the "Hello, World!" message.

- Please create a component that uses the lightning-card component to display information.

         // a simple example of an LWC that displays Employee Information within a card: 
        
         <!-- cardExample.html --> 
        

        <!-- cardExample.html -->
        <template>
          <lightning-card title="Employee Information" icon-name="standard:contact">
              <div class="slds-p-horizontal_small">
                  <p>Name: {employeeName}</p>
                  <p>Title: {employeeTitle}</p>
                  <p>Email: {employeeEmail}</p>
              </div>
          </lightning-card>
        </template>
        
  
         // cardExample.js 
        

          // cardExample.js
          import { LightningElement } from 'lwc';
          
          export default class CardExample extends LightningElement {
              employeeName = "Rose Blossom";
              employeeTitle = "Salesforce Developer";
              employeeEmail = "roseblossom@example.com";
          }          
        

        
            // In this example, we have an LWC named "CardExample" that uses the lightning-card component to create a card with employee information.
            // The title attribute of lightning-card sets the card's title, and the icon-name attribute sets the card's icon.
            // Inside the card, we have a simple template that displays the employee's name, title, and email using {} syntax, which binds the values to the component's properties.
            // We can include this LWC in our Salesforce Lightning App to display the employee information within a card component.

- Please create a component that creates a basic form for collecting user information:

         // We uses lightning-input tags to create a form. 
        
         <!-- helloWorld.html --> 
        

        <!-- inputExample.html -->
<template>
    <lightning-card title="User Information">
        <div class="slds-p-horizontal_small">
            <lightning-input label="First Name" value={firstName} onchange={handleFirstNameChange}></lightning-input>
            <lightning-input label="Last Name" value={lastName} onchange={handleLastNameChange}></lightning-input>
            <lightning-input label="Email" type="email" value={email} onchange={handleEmailChange}></lightning-input>
        </div>
    </lightning-card>
</template>
        
  
         // helloWorld.js 
        

        // inputExample.js
import { LightningElement } from 'lwc';

export default class InputExample extends LightningElement {
    firstName = '';
    lastName = '';
    email = '';

    handleFirstNameChange(event) {
        this.firstName = event.target.value;
    }

    handleLastNameChange(event) {
        this.lastName = event.target.value;
    }

    handleEmailChange(event) {
        this.email = event.target.value;
    }
}
        

        
          // In this example, we have an LWC named "InputExample" that uses lightning-input tags to create a form for collecting user information.
          // Each lightning-input component has a label attribute for setting the input label and an onchange event handler to update the corresponding property when the user enters data.
          // The @track decorator is used to make the properties reactive, so changes in the input fields are reflected in the component's state.
          // We can include this LWC in our Salesforce Lightning App to create a simple user information form.
